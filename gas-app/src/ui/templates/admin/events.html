Event Management

info_outline
Creating an event only makes the Drive folder. Before anyone can upload photos, you must create an upload link for the event — use the "Create Upload Link" button on each row below. To view or manage the links that already exist for an event, open the Upload Links page and select the event.
创建活动只会生成 Drive 文件夹。在任何人上传照片之前，您必须为该活动创建上传链接—— 点击下方每行的「Create Upload Link / 创建上传链接」按钮。 要查看或管理某个活动已有的链接，请打开上传链接页面并选择该活动。
