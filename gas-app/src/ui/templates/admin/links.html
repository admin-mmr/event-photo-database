link Upload Link Management 上传链接管理

info_outline
Each upload link is scoped to one event + club pair, with an optional tag (e.g. "finish_line", "mile_10") to separate photos by photographer location. Leave tag empty to create a single shared link — all photos go into one folder. Links are permanent and bearer-token based — anyone with the URL and a Google account can upload. Rotate a link to invalidate the old URL; all prior audit records remain intact.
每条上传链接绑定一个赛事 + 俱乐部组合，可附加可选的标签（如"终点线"、"10公里处"）以按摄影师位置分类照片。 不填标签则所有照片进入同一相册。链接永久有效，基于Bearer令牌——任何持有链接且有Google账号的人均可上传。 轮换链接可使旧URL失效，所有历史审计记录保持完整。

link_off Revoke Upload Link / 撤销上传链接

Holders of the current URL will immediately receive a "link revoked" message. The event–club pair is preserved; generate a new link afterward if needed.
当前URL的持有者将立即收到"链接已撤销"提示。赛事-俱乐部关联保留，如需可重新生成新链接。

autorenew Rotate Upload Link / 轮换上传链接

The current URL will be invalidated and a new one generated for the same event and club. All prior audit records remain intact — the rotation is noted in the audit log.
当前URL将失效，同一赛事与俱乐部将生成新链接。所有历史审计记录保持完整，本次轮换操作将记入审计日志。
